Explicación del proyecto:
El  proyecto consiste en un sistema de detección cuando una persona esta usando cubrebocas. En caso de que la persona no este usando cubrebocas se almacenara esta imagen en una base de datos local. Esta base de datos solo guardara fotos a personas sin cubrebocas, mientras a las que están portando cubrebocas se descartaran.

Como objetivos futuros se pretende implementar la base de datos en lo que seria la nube de IBM, se pretende implementar el servicio DB2.
Ademas de por falta de material, no se pudo implementar en una tarjeta de desarrollo Raspberry pi 3,  pero resultado el mismo proceso solo sonando una alarma cuando detecta a la persona sin cubrebocas, incluso implementado el servicio de IoT de IBM.
